# Multilingual Metaphor Translator Pro

## 📌 Descripción
Este proyecto fue desarrollado en [PartyRock](https://partyrock.aws/), un entorno de **IA generativa** creado por AWS.  
La aplicación se construyó mediante **instrucciones específicas (prompts)** dirigidas a **Whiskers**, el asistente de PartyRock, aplicando técnicas de *prompt engineering* para diseñar un flujo de widgets que generan metáforas, traducciones multilingües, definiciones técnicas, historias creativas e imágenes ilustrativas.  

Este trabajo forma parte del curso **“Practitioner AI Challenge”** de la plataforma educacional **Udacity**, realizado en el año **2026** como parte del programa de becas impulsado por AWS.  

El enfoque demuestra habilidades en:
- **Prompt engineering** (diseño de instrucciones claras y estructuradas).  
- **Aplicación práctica de IA generativa sin código**.  
- **Integración de múltiples salidas en una sola app** (texto, imagen, historial).  

---

## 🛠️ Guía de uso
**Cómo usar la aplicación:**
- **Concepto técnico:** Ingresa el concepto que quieres entender mediante una metáfora.  
- **Idioma objetivo:** Selecciona uno o varios idiomas para la traducción.  
- **Calificación de claridad:** Usa el deslizador (1–5). La valoración se guarda en el historial.  
- **Explicación metafórica:** Genera una metáfora simple usando situaciones cotidianas.  
- **Definición técnica:** Proporciona una explicación concisa del concepto.  
- **Historia o poema creativo:** Convierte la metáfora en un relato o poema.  
- **Opciones de compartir:** Elige versión formal o creativa para copiar y compartir.  
- **Imagen del concepto:** Genera una ilustración del concepto.  
- **Historial de metáforas:** Guarda metáforas pasadas con calificación y fecha.
